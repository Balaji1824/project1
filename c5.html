HTML Attributes

HTML attributes provide additional information about HTML elements.

  • All HTML elements can have attributes
  • Attributes provide additional information about elements
  • Attributes are always specified in the start tag
  • Attributes usually come in name/value pairs like: name="value"

The href Attribute

The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to:

<a href="https://www.w3schools.com">Visit W3Schools</a>

The src Attribute

The <img> tag is used to embed an image. The src attribute specifies the image path:

<img src="img_girl.jpg">

URL Types:

  • Absolute URL: src="https://www.w3schools.com/images/img_girl.jpg"
  • Relative URL: src="img_girl.jpg" or src="/images/img_girl.jpg"

The width and height Attributes

<img src="img_girl.jpg" width="500" height="600">

The alt Attribute

<img src="img_girl.jpg" alt="Girl with a jacket">

If the image can't load, the alt text is shown.

<img src="img_typo.jpg" alt="Girl with a jacket">

The style Attribute

<p style="color:red;">This is a red paragraph.</p>

The lang Attribute

<!DOCTYPE html>
<html lang="en">
<body>
...
</body>
</html>

With country code:

<html lang="en-US">

The title Attribute

<p title="I'm a tooltip">This is a paragraph.</p>

Use Lowercase and Quoted Attribute Values

Recommended:

<a href="https://www.w3schools.com/html/">Visit our HTML tutorial</a>

Not recommended:

<a href=https://www.w3schools.com/html/>Visit our HTML tutorial</a>

Single vs Double Quotes

<p title='John "ShotGun" Nelson'>
<p title="John 'ShotGun' Nelson">

Summary

  • All HTML elements can have attributes
  • href in <a> links to a URL
  • src in <img> points to an image
  • width and height control image size
  • alt gives alternate image text
  • style adds CSS styles
  • lang declares page language
  • title shows extra info as tooltip
