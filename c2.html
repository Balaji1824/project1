HTML Editors

Learn how to write and edit HTML using simple tools

Learn HTML Using Notepad or TextEdit

Web pages can be created and modified by using professional HTML editors. However, for learning HTML we recommend a simple text editor like Notepad (PC) or TextEdit (Mac).

Step 1: Open Notepad (PC)

  • Windows 8 or later: Open the Start Screen, type Notepad.
  • Windows 7 or earlier: Open Start > Programs > Accessories > Notepad.

Step 1: Open TextEdit (Mac)

  • Open Finder > Applications > TextEdit.
  • Set Preferences > Format > choose "Plain Text".
  • In "Open and Save", check "Display HTML files as HTML code instead of formatted text".

Step 2: Write Some HTML

Write or copy the following HTML code into Notepad:

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>

<p>My first paragraph.</p>

</body>
</html>

Step 3: Save the HTML Page

Save the file as index.htm and set encoding to UTF-8.

Step 4: View the HTML Page in Your Browser

Open the saved file in your browser to see the result.

Use Online Editor

Use the online editor to edit and test HTML code easily.

<!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>
