HTML Basic Examples

Let's explore basic HTML usage through examples

HTML Documents

All HTML documents must start with a document type declaration: <!DOCTYPE html>.

The HTML document itself begins with <html> and ends with </html>.

The visible part of the HTML document is between <body> and </body>.

Example

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

The <!DOCTYPE> Declaration

The <!DOCTYPE> declaration helps browsers display web pages correctly and must appear only once at the top of the page.

HTML Headings

HTML headings are defined with the <h1> to <h6> tags:

<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>

HTML Paragraphs

Paragraphs are defined with the <p> tag:

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

HTML Links

Links are defined with the <a> tag and the href attribute specifies the destination:

<a href="https://www.w3schools.com">This is a link</a>

HTML Images

Images are defined with the <img> tag:

<img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">

How to View HTML Source

To view a page's HTML source code, press CTRL + U or right-click and select "View Page Source".

To inspect an element, right-click it and choose "Inspect" to view and edit its HTML and CSS live.
