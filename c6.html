HTML Headings

What are HTML Headings?

HTML headings are titles or subtitles that you want to display on a webpage.

Example:

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

HTML Heading Tags

HTML headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading and <h6> defines the least important.

Note:

Browsers automatically add some white space (a margin) before and after a heading.

Why Are Headings Important?

Search engines use headings to index the structure and content of your web pages. Users often skim a page by its headings, so use them to show your document's structure properly.

<h1> headings should be used for main headings, followed by <h2>, then the less important <h3>, and so on.

Tip:

Use HTML headings only for headings. Don't use them to make text big or bold.

Bigger Headings with CSS

Each HTML heading has a default size. However, you can customize it using the style attribute with the CSS font-size property:

<h1 style="font-size:60px;">Heading 1</h1>
